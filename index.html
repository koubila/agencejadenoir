Comment utiliser ces logos

Ces concepts de logo sont fournis en format SVG, parfaitement adaptés pour le web.

Vous pouvez copier le code SVG de chaque logo et l'intégrer directement dans votre site web.

Pour une utilisation professionnelle, nous recommandons de faire finaliser le logo par un graphiste qui pourra :

- Créer des versions haute résolution

- Développer un guide d'utilisation complet

- Adapter le logo pour tous supports (print, textile, etc.)
